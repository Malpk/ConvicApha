Как добавить новый предмет в меню.
Нужно создать префаб UI вьюшки.
Этот префаб состоит из компонента Image и Компонента, наследующего  BaseScrollElementView.
Для добавления Предмета, нужно добавить компонент ItemView, и прокинуть префаб игрового предмета в поле itemPrefab.

Получившийся префаб UI предмета кидаем в список "ListPrefabs" того Scroller'a, куда нужно.

Для добавления нового Персонажа, также создаем префаб UI персонажа.

Все то же самое, только вместо ItemView, CharacterView.
В CharacterView выбираем PlayerType и Описание игрока(Description).
Description это описание игрока в меню. Создается в меню.Assets->Create->Create Description Character
Добавляем UI префаб игрока в "ListPrefabs" Скроллера персонажей.

Если кто-то захочет новый скроллер для нового типа чего-то, то нужно будет 
написать свои компоненты на основе базовых.
Базовый скроллер включает в себя 2 кнопки, которые автоматически подписываются на методы Next, Previous.
_transforms это точки, по которым перемещаются вьюшки.
_listPrefabs собственно список префабов вьюшек, из которых потом спавнятся вьюшки.
_parentElements - родитель, под которого спавнятся вьюшки.
